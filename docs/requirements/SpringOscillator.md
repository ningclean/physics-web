# 物理场景需求：弹簧振子 (Spring Oscillator)

### 1. 场景名称
**名称**：弹簧振子 (Spring Oscillator)

### 2. 物理过程描述
**描述**：一个连接在弹簧上的质点，在弹性回复力和阻尼力作用下做振动。支持水平和竖直两种模式。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 模式 | - | - | 水平 | - | 选择水平振子或竖直振子 |
| 质量 | $m$ | 0.5 - 5.0 | 1.0 | 0.1 | 小球的质量 (kg) |
| 劲度系数 | $k$ | 10 - 100 | 20 | 5 | 弹簧的硬度 (N/m) |
| 阻尼系数 | $c$ | 0.0 - 5.0 | 0.5 | 0.1 | 阻尼系数 |
| 振幅 | $A$ | 50 - 250 | 150 | 10 | 初始拉伸距离 (px) |
| 重力 | $g$ | 100 - 1000 | 500 | 50 | 重力加速度 (仅竖直模式) |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示回复力 | - | true/false | true | - | 开关：显示弹簧弹力 |
| 显示阻尼力 | - | true/false | true | - | 开关：显示阻尼力 |
| 显示速度 | - | true/false | true | - | 开关：显示速度矢量 |
| 显示重力 | - | true/false | true | - | 开关：显示重力矢量 |

### 4. 视觉元素与图例

*   **主要物体**：小球 (圆形)，弹簧 (锯齿线)。
*   **环境物体**：墙壁/天花板，地面，平衡位置参考线。
*   **矢量箭头**：
    *   [x] 显示速度 (颜色：橙色)
    *   [x] 显示回复力 (颜色：绿色)
    *   [x] 显示阻尼力 (颜色：红色)
    *   [x] 显示重力 (颜色：紫色)

### 5. 数学公式

*   **公式 1**：动力学方程
    $$ F_{net} = -kx - cv = ma $$
*   **公式 2**：固有角频率
    $$ \omega_n = \sqrt{\frac{k}{m}} $$
*   **公式 3**：平衡位置伸长量 (竖直模式)
    $$ \Delta l = \frac{mg}{k} $$
