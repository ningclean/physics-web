# 物理场景需求：简谐运动 (SHM)

### 1. 场景名称
**名称**：简谐运动 (Simple Harmonic Motion)

### 2. 物理过程描述
**描述**：一个质点在平衡位置附近做往复运动，其回复力与偏离平衡位置的位移成正比且方向相反。这里展示的是水平方向的简谐运动投影。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 角频率 | $\omega$ | 0.1 - 5.0 | 1.0 | 0.1 | 控制振动的快慢 |
| 振幅 | $A$ | 50 - 250 | 150 | 10 | 偏离平衡位置的最大距离 (px) |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示速度 | - | true/false | true | - | 开关：显示速度矢量 |
| 显示加速度 | - | true/false | true | - | 开关：显示加速度矢量 |
| 显示轨迹 | - | true/false | true | - | 开关：显示运动路径 |

### 4. 视觉元素与图例

*   **主要物体**：质点 (圆形)。
*   **环境物体**：运动范围参考线 (虚线)，坐标轴。
*   **矢量箭头**：
    *   [x] 显示速度 (颜色：橙色)
    *   [x] 显示加速度 (颜色：蓝色，指向平衡位置)
*   **轨迹**：
    *   [x] 显示运动轨迹 (颜色：根据背景自适应)

### 5. 数学公式

*   **公式 1**：位移
    $$ x(t) = A \cos(\omega t) $$
*   **公式 2**：速度
    $$ v_x(t) = -A\omega \sin(\omega t) $$
*   **公式 3**：加速度
    $$ a_x(t) = -A\omega^2 \cos(\omega t) $$
