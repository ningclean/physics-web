# 物理场景需求：自由落体 (Free Fall)

### 1. 场景名称
**名称**：自由落体 (Free Fall)

### 2. 物理过程描述
**描述**：物体只受重力作用，从静止或初速度开始竖直下落。本场景包含地面碰撞模拟，支持设置弹性系数。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 初始高度 | $H$ | 50 - 400 | 300 | 10 | 起始距离地面的高度 (px) |
| 重力加速度 | $g$ | 1 - 30 | 9.8 | 0.1 | 环境重力大小 |
| 初速度 | $v_0$ | -20 - 20 | 0 | 1 | 垂直方向初速度 (负值向上) |
| 弹性系数 | $e$ | 0 - 1 | 0 | 0.1 | 地面反弹损耗 (0不反弹，1完全弹性) |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示速度 | - | true/false | true | - | 开关：显示速度矢量 |
| 显示加速度 | - | true/false | true | - | 开关：显示重力加速度矢量 |
| 显示受力 | - | true/false | true | - | 开关：显示重力矢量 |
| 显示轨迹 | - | true/false | true | - | 开关：显示运动路径 |

### 4. 视觉元素与图例

*   **主要物体**：小球 (红色圆形)。
*   **环境物体**：地面 (线条和斜线纹理)。
*   **矢量箭头**：
    *   [x] 显示速度 (颜色：橙色)
    *   [x] 显示加速度 (颜色：蓝色)
    *   [x] 显示受力 (颜色：绿色)
*   **轨迹**：
    *   [x] 显示运动轨迹 (颜色：根据背景自适应)

### 5. 数学公式

*   **公式 1**：位移公式
    $$ y(t) = H - (v_0 t + \frac{1}{2}gt^2) $$
*   **公式 2**：速度公式
    $$ v(t) = v_0 + gt $$
*   **参数解释**：
    *   $y$: 高度
    *   $H$: 初始高度
    *   $v_0$: 初速度
    *   $g$: 重力
