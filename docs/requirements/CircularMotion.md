# 物理场景需求：匀速圆周运动 (Circular Motion)

### 1. 场景名称
**名称**：匀速圆周运动 (Circular Motion)

### 2. 物理过程描述
**描述**：一个质点绕固定圆心做匀速圆周运动。速度大小不变，方向时刻改变。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 角速度 | $\omega$ | 0.1 - 5.0 | 0.5 | 0.1 | 控制转动的快慢 (rad/s) |
| 半径 | $R$ | 50 - 250 | 150 | 10 | 圆周运动的轨道半径 (px) |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示速度 | - | true/false | true | - | 开关：显示速度矢量 |
| 显示加速度 | - | true/false | true | - | 开关：显示向心加速度矢量 |
| 显示轨迹 | - | true/false | true | - | 开关：显示运动路径 |

### 4. 视觉元素与图例

*   **主要物体**：质点 (圆形)。
*   **环境物体**：圆形轨道 (线条)，坐标轴。
*   **矢量箭头**：
    *   [x] 显示速度 (颜色：橙色，沿切线方向)
    *   [x] 显示加速度 (颜色：蓝色，指向圆心)
*   **轨迹**：
    *   [x] 显示运动轨迹 (颜色：根据背景自适应)

### 5. 数学公式

*   **公式 1**：角度随时间变化
    $$ \theta(t)=\omega t $$
*   **公式 2**：位置矢量
    $$ \vec{r}(t) = R(\cos\theta \hat{i} + \sin\theta \hat{j}) $$
*   **参数解释**：
    *   $\theta$: 角度
    *   $\omega$: 角速度
    *   $R$: 半径
