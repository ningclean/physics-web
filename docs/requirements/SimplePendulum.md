# 物理场景需求：单摆 (Simple Pendulum)

### 1. 场景名称
**名称**：单摆 (Simple Pendulum)

### 2. 物理过程描述
**描述**：一个质点（摆球）悬挂在不可伸长的轻绳下端，在重力作用下在竖直平面内做往复摆动。模拟中包含空气阻力（阻尼）。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 摆长 | $L$ | 0.5 - 3.0 | 2.0 | 0.1 | 绳子的长度 (m) |
| 初始角度 | $\theta_0$ | -90 - 90 | 30 | 5 | 初始偏离竖直方向的角度 (度) |
| 质量 | $m$ | 0.1 - 5.0 | 1.0 | 0.1 | 摆球的质量 (kg) |
| 重力 | $g$ | 1.0 - 20.0 | 9.8 | 0.1 | 重力加速度 (m/s²) |
| 阻尼系数 | $c$ | 0.0 - 2.0 | 0.5 | 0.1 | 空气阻力系数 |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示速度 | - | true/false | true | - | 开关：显示速度矢量 |
| 显示受力 | - | true/false | true | - | 开关：显示重力和拉力 |

### 4. 视觉元素与图例

*   **主要物体**：摆球 (圆形)，绳子 (线条)，悬挂点 (小圆点)。
*   **环境物体**：竖直参考虚线。
*   **矢量箭头**：
    *   [x] 显示速度 (颜色：橙色)

    *   [x] 显示受力：

        *   拉力 (颜色：绿色)
*   **轨迹**：
    *   [ ] 显示运动轨迹 (未实现)

### 5. 数学公式

*   **公式 1**：运动方程
    $$ \frac{d^2\theta}{dt^2} = -\frac{g}{L}\sin\theta - \frac{c}{m}\frac{d\theta}{dt} $$
*   **公式 2**：周期 (小角度)
    $$ T \approx 2\pi\sqrt{\frac{L}{g}} $$
*   **参数解释**：
    *   $\theta$: 摆角
    *   $L$: 摆长
    *   $c$: 阻尼系数
