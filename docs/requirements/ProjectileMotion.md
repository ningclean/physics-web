# 物理场景需求：平抛/斜抛运动 (Projectile Motion)

### 1. 场景名称
**名称**：平抛/斜抛运动 (Projectile Motion)

### 2. 物理过程描述
**描述**：物体以一定的初速度和角度抛出，在重力（和可选的空气阻力）作用下做曲线运动。展示运动的独立性原理（水平方向匀速/减速，竖直方向匀变速）。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 初速度 | $v_0$ | 10 - 100 | 50 | 1 | 发射时的速度大小 (m/s) |
| 发射角度 | $\theta$ | 0 - 90 | 45 | 1 | 发射方向与水平面的夹角 (度) |
| 初始高度 | $h$ | 0 - 200 | 0 | 10 | 发射点距离地面的高度 (m) |
| 重力 | $g$ | 1.0 - 20.0 | 9.8 | 0.1 | 重力加速度 (m/s²) |
| 空气阻力 | $k$ | 0.0 - 1.0 | 0.0 | 0.01 | 空气阻力系数 (与速度成正比或平方) |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示速度 | - | true/false | true | - | 开关：显示合速度矢量 |
| 显示分速度 | - | true/false | true | - | 开关：显示水平和垂直分速度 |
| 显示轨迹 | - | true/false | true | - | 开关：显示运动路径 |

### 4. 视觉元素与图例

*   **主要物体**：炮弹/小球 (圆形)。
*   **环境物体**：地面，发射台 (可选)。
*   **矢量箭头**：
    *   [x] 显示合速度 (颜色：橙色)
    *   [x] 显示水平分速度 $v_x$ (颜色：淡橙色/虚线)
    *   [x] 显示垂直分速度 $v_y$ (颜色：淡橙色/虚线)
    *   [x] 显示加速度/重力 (颜色：蓝色)
*   **轨迹**：
    *   [x] 显示抛物线轨迹

### 5. 数学公式

*   **公式 1**：水平位移 (无阻力)
    $$ x(t) = v_0 \cos\theta \cdot t $$
*   **公式 2**：垂直位移 (无阻力)
    $$ y(t) = h + v_0 \sin\theta \cdot t - \frac{1}{2}gt^2 $$
*   **公式 3**：轨迹方程
    $$ y = x \tan\theta - \frac{g}{2v_0^2\cos^2\theta}x^2 + h $$
