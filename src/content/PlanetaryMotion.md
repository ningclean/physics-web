# 物理场景需求：天体运动 (Planetary Motion)

### 1. 场景名称
**名称**：天体运动 (Planetary Motion)

### 2. 物理过程描述
**描述**：模拟行星绕恒星的运动，展示万有引力定律和开普勒定律。支持调整初速度以形成椭圆、圆形或双曲线轨道。

### 3. 可调节参数 (UI控件)

| 参数名 | 符号 | 范围 (Min - Max) | 默认值 | 步长 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 恒星质量 | $M$ | 100 - 1000 | 500 | 50 | 中心天体质量 (相对单位) |
| 行星初速度 | $v_0$ | 0 - 20 | 10 | 0.5 | 行星切向初速度 |
| 初始距离 | $r$ | 100 - 400 | 200 | 10 | 行星与恒星的初始距离 |
| 引力常数 | $G$ | 0.1 - 5.0 | 1.0 | 0.1 | 模拟引力常数 |
| 矢量缩放 | - | 0.1 - 3.0 | 1.0 | 0.1 | 调整矢量箭头的显示长度 |
| 显示速度 | - | true/false | true | - | 开关：显示速度矢量 |
| 显示引力 | - | true/false | true | - | 开关：显示万有引力矢量 |
| 显示轨迹 | - | true/false | true | - | 开关：显示轨道 |

### 4. 视觉元素与图例

*   **主要物体**：恒星 (发光大球)，行星 (小球)。
*   **环境物体**：星空背景 (可选)。
*   **矢量箭头**：
    *   [x] 显示速度 (颜色：橙色，沿切线)
    *   [x] 显示引力 (颜色：紫色，指向恒星)
*   **轨迹**：
    *   [x] 显示完整的轨道轨迹 (闭合椭圆或开放曲线)

### 5. 数学公式

*   **公式 1**：万有引力定律
    $$ \vec{F} = -G \frac{Mm}{r^2} \hat{r} $$
*   **公式 2**：开普勒第二定律
    $$ \frac{dA}{dt} = \text{const} $$
    (掠面速度守恒)
*   **公式 3**：机械能守恒
    $$ E = \frac{1}{2}mv^2 - G\frac{Mm}{r} $$
