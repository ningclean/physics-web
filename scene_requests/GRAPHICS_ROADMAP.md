# 图形组件库规划 (Graphics Library Roadmap)

为了提升物理引擎的视觉效果和复用性，计划在 `src/utils/graphics.js` 中扩展以下组件。

## 1. 力学基础组件 (Mechanics)

### 1.1 弹簧 (Spring)
*   **描述**: 螺旋状线条，支持拉伸和压缩。
*   **参数**: 起点(x1, y1), 终点(x2, y2), 宽度(width), 圈数(coils), 颜色(color)。
*   **用途**: 弹簧振子、胡克定律演示。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 1.2 阻尼器 (Damper / Dashpot)
*   **描述**: 由活塞和圆筒组成的阻尼装置。
*   **参数**: 起点(x1, y1), 终点(x2, y2), 宽度(width), 活塞位置(extension), 颜色(color)。
*   **用途**: 阻尼振动、汽车悬挂模拟。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 1.3 支架 / 悬挂点 (Support / Pivot)
*   **描述**: 固定点标识，通常为三角形或矩形底座，带有表示固定的斜线阴影。
*   **参数**: 位置(x, y), 宽度(width), 类型(type: 'ceiling' | 'floor' | 'wall'), 颜色(color)。
*   **用途**: 单摆悬挂点、弹簧固定端、杠杆支点。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 1.4 实验小车 (Lab Cart)
*   **描述**: 带有两个轮子的矩形车身，比普通滑块更具物理实验感。
*   **参数**: 中心位置(x, y), 宽度(width), 高度(height), 轮子半径(wheelRadius), 颜色(color)。
*   **用途**: 牛顿第二定律、动量守恒（碰撞）、斜面运动。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 1.5 滑轮 (Pulley)
*   **描述**: 圆形轮盘，带有中心轴和边缘槽，可能包含悬挂支架。
*   **参数**: 中心位置(x, y), 半径(radius), 颜色(color)。
*   **用途**: 阿特伍德机、滑轮组。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

## 2. 测量与辅助工具 (Measurement & Tools)

### 2.1 标尺 (Ruler)
*   **描述**: 带有刻度的长条尺。
*   **用途**: 测量距离、展示坐标系。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 2.2 量角器 (Protractor)
*   **描述**: 半圆形角度测量工具。
*   **用途**: 光学角度测量、单摆摆角。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 2.3 秒表 (Stopwatch)
*   **描述**: 模拟或数字显示的计时器。
*   **用途**: 记录运动时间。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

## 3. 电磁学组件 (Electromagnetism)

### 3.1 条形磁铁 (Bar Magnet)
*   **描述**: 经典的红蓝配色长方体，标有 N 和 S。
*   **参数**: 中心位置(x, y), 宽度(width), 高度(height), 角度(angle)。
*   **用途**: 磁场演示、电磁感应。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 3.2 点电荷 (Point Charge)
*   **描述**: 带正(+)或负(-)符号的球体，具有发光晕染效果表示电场。
*   **参数**: 位置(x, y), 半径(radius), 电荷量(charge, 正负决定颜色和符号)。
*   **用途**: 库仑定律、电场线模拟。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 3.3 电池 (Battery)
*   **描述**: 干电池外观，标有正负极。
*   **参数**: 位置(x, y), 宽度(width), 高度(height), 角度(angle)。
*   **用途**: 电路基础演示。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 3.4 灯泡 (Light Bulb)
*   **描述**: 传统的白炽灯泡外观，支持点亮(发光)和熄灭状态。
*   **参数**: 位置(x, y), 大小(size), 状态(isOn), 颜色(color)。
*   **用途**: 电路通断演示。
*   **状态**: *已实现* (已迁移至 `src/utils/graphics.js`)。

### 3.5 电线 (Wire)
*   **描述**: 连接电路元件的导线，可带绝缘皮颜色。
*   **参数**: 路径点(points), 颜色(color), 线宽(width)。
*   **用途**: 连接电路。

### 3.6 开关 (Switch)
*   **描述**: 单刀单掷开关，支持断开和闭合状态。
*   **参数**: 位置(x, y), 宽度(width), 状态(isOpen)。
*   **用途**: 控制电路通断。

### 3.7 万用表 (Multimeter)
*   **描述**: 数字万用表，显示电压、电流或电阻数值。
*   **参数**: 位置(x, y), 宽度(width), 数值(value), 单位(unit)。
*   **用途**: 测量电路参数。

### 3.8 电阻 (Resistor)
*   **描述**: 带有色环的圆柱体电阻。
*   **参数**: 位置(x, y), 宽度(width), 色环颜色(bands)。
*   **用途**: 电路限流元件。
